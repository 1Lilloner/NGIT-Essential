				TASK 4
The website visited is www.mathworks.com
The domain name is mathworks.com
It uses HTTPS
What happens behind the scenes: 
1.  DNS Lookup: The browser sends a request to a DNS server to "translate" the name mathworks.com into a numerical IP address.
2.  Handshake: The computer establishes a secure connection with the MathWorks server using the HTTPS protocol to ensure your data stays private.
3.  Request/Response: The browser sends a GET request for the homepage; the server finds the files and sends back a package of HTML, CSS, and JavaScript.
4.  Rendering: Your browser’s engine reads that code and paints the "Welcome back" screen displayed.

DNS Lookup: The browser sends a request to a DNS server to translate mathworks.com into an IP address.

Handshake: A secure connection is established with the server using HTTPS to ensure data is encrypted.

Request/Response: The browser sends a request for the webpage, and the server responds with HTML, CSS, and JavaScript files.

Rendering: The browser processes the files, builds the page structure, and displays the website on the screen.
